# 📖 Project Overview

Industria AI 2.0 is an AI-powered Enterprise Industrial Intelligence Platform developed to solve **Problem Statement 8** of the ET AI Hackathon.

The platform transforms unstructured industrial documents into structured knowledge using Artificial Intelligence, OCR, Entity Extraction, and Intelligent Analytics, enabling engineers and decision-makers to reduce downtime, improve compliance, and make faster data-driven decisions.

---

# 🎯 Problem Statement

Industrial organizations store valuable knowledge across:

- Maintenance Reports
- Inspection Reports
- Equipment Manuals
- Safety Documents
- Compliance Reports
- Operational Logs

Finding relevant information is slow, disconnected, and inefficient.

Industria AI 2.0 solves this challenge by converting industrial documents into an intelligent knowledge system powered by AI.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/MuthyalaVaraprasad/Industrial-AIHackathon.git
```

## Navigate

```bash
cd Industrial-AIHackathon
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```
